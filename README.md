# 🎬 Movie Search App

A simple and responsive Movie Search Application built using React. This app allows users to search for movies and view details like title, year, type, and poster using the OMDb API.

---

## 🚀 Features

* 🔍 Search movies by name
* 🎬 View movie details (Title, Year, Type, Poster)
* ⚡ Fast and responsive UI
* 🎨 Modern and attractive design

---

## 🛠️ Technologies Used

* React (Vite)
* JavaScript (ES6)
* CSS (Custom Styling)
* Axios
* OMDb API

---

## 🔑 API Used

This project uses the OMDb API to fetch movie data.

Example API:
http://www.omdbapi.com/?s=batman&apikey=YOUR_API_KEY

## ▶️ How to Run the Project

1. Clone the repository:

```
git clone https://github.com/Arshad-sh/movie-search.git
```

2. Navigate to project folder:

```
cd movie-search
```

3. Install dependencies:

```
npm install
```

4. Start the development server:

```
npm run dev
```
